# IPL Data Analysis
Exploratory data analysis on IPL (2008–2024) dataset using Python.

## Tools Used
- Python (Pandas, Matplotlib, Seaborn)
- Jupyter Notebook
- Dataset: [Kaggle IPL Dataset](https://www.kaggle.com/datasets/vora1011/ipl-2008-to-2021-all-match-dataset)

## What I Analyzed
- Team win records
- Toss impact on match results
- Top batsmen and bowlers
- Season-wise trends
